Quality Computer Battles

Quality Computer Battles is a simple dungeon-crawling shoot-em-up game themed around fixing bugs in a computer as a QA tester.

This was a short project, about 4 weeks long. It was used as an introduction to game engines and making games in the C language

This project is unavailable for public download.
